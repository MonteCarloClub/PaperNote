# PaperNote

> 大家读完论文之后，要思考，这篇论文的优点缺点是什么、能不能做广一点深一点、可以用在哪些场景，等等，每一篇论文都是不完善的，可以继续思考的。

> 不能读完了就结束了，完成任务一样，这样没有什么效果的

## 〇、说明

`PaperNote`
用来存储我们读过的论文及笔记、学术会议记录等。

`papers`
目录存储论文原文，根据类别整理。

`notes`
目录存储笔记，建议写成Markdown格式。

其他目录根据需要创建。

以下内容请分类整理，例如物流算法、密码学、……
